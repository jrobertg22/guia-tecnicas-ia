Guía de Estudio: Técnicas de Inteligencia Artificial

Tema 03: Python para la implementación de técnicas de inteligencia artificial

2.2. Introducción

  • Origen y Popularidad: Python es un lenguaje creado por Guido van Rossum y lanzado en 1991. Actualmente es el lenguaje más empleado en el ámbito de la inteligencia artificial y el machine learning, con un 57% de científicos de datos utilizándolo.
  • Motivos de su éxito: Su popularidad se disparó por el lanzamiento de bibliotecas como TensorFlow por Google en 2015. Además, cuenta con una sintaxis muy simple, lo que lo hace idóneo para principiantes y permite que los algoritmos sean fáciles de implementar.

2.3. El lenguaje Python: conceptos básicos e instalación

Características Clave del Lenguaje:
  • Interpretado: Ejecuta sentencias línea a línea sin necesidad de compilación previa (como ocurre en C/C++ o Pascal).
  • Multiparadigma: Soporta programación orientada a objetos, imperativa/procedural y funcional.
  • Tipado: Es fuertemente tipado (no permite violaciones de tipos de datos sin conversión explícita) y de tipado dinámico (una variable puede cambiar de tipo en tiempo de ejecución).

Instalación: Se puede instalar desde su web oficial (que incluye el gestor de paquetes PIP) o mediante gestores de entornos como Conda o Miniconda, muy útiles en el análisis de datos.

2.4. La sintaxis de Python

  • Indentación estricta: El sangrado es obligatorio para definir los distintos bloques de ejecución.
  • Variables: Se crean automáticamente al usarlas por primera vez, son sensibles a mayúsculas/minúsculas (case-sensitive) y no pueden comenzar por un número.
  • Operadores especiales: Además de los aritméticos y lógicos, destacan los operadores de identidad (is, is not) que verifican si dos referencias apuntan a la misma memoria, y los de membresía (in, not in) para iterar o buscar en colecciones.

2.5. Listas, tuplas, conjuntos y diccionarios

Python incluye cuatro tipos básicos de colecciones para manejar datos:

  • Listas ([]): Colecciones ordenadas y modificables. Permiten duplicados y elementos de diferentes tipos.
  • Tuplas (()): Colecciones ordenadas pero inmutables (no se puede alterar su valor tras crearlas).
  • Conjuntos / Sets ({}): Colecciones no ordenadas que no permiten elementos duplicados ni acceso por índice.
  • Diccionarios ({clave: valor}): Equivalentes a mapas o hashes. Son pares clave-valor sin orden, flexibles y modificables.

2.6. Librerías útiles para el análisis de datos

El ecosistema de Python es rico en librerías esenciales para IA:

  • NumPy: Procesamiento eficiente de matrices multidimensionales y álgebra lineal.
  • SciPy: Módulos para optimización, integración y estadística avanzada.
  • Scikit-learn: Librería principal para algoritmos de machine learning clásico (supervisado y no supervisado).
  • Pandas: Ideal para la extracción, limpieza y preparación de datos utilizando dataframes.
  • Matplotlib: Librería de trazado 2D para visualización de patrones de datos e histogramas.
  • Deep Learning: Keras (API de alto nivel para redes neuronales), TensorFlow (cálculo de tensores de alto rendimiento de Google), y PyTorch (cálculo tensorial con aceleración GPU).

2.7. La librería NumPy para el manejo de datos

  • Rendimiento: Proporciona arrays (darray) unas 50 veces más rápidos que las listas nativas debido a su almacenamiento contiguo en memoria y su escritura interna en C/C++.
  • Copy vs View: copy() genera un array nuevo e independiente, mientras que view() crea una referencia que, al modificarse, altera los datos del original.
  • Manipulación: Ofrece gran flexibilidad con el slicing (particionado), redimensionamiento (reshape()) y filtros de búsqueda mediante where().

2.8. Importación de datos

Es un paso clave para nutrir modelos predictivos. Opciones destacadas:

  • Librería estándar: A través del módulo csv, leyendo línea por línea.
  • Pandas (Recomendado): La función read_csv() es altamente versátil; detecta automáticamente cabeceras, infiere tipos de datos, maneja campos vacíos y convierte el conjunto en un dataframe eficiente.
  • Fuentes Externas y "Toy Datasets": Carga directa desde URLs (ej. repositorios OpenML o Kaggle) o utilizando conjuntos de prueba ya incluidos en librerías como Scikit-learn.

2.9. Introducción a Machine Learning con librerías en Python

  • Estadística Descriptiva: Uso de NumPy y SciPy para obtener de manera rápida la media, mediana, moda, varianza y percentiles de variables continuas, discretas o categóricas.
  • Distribución de datos: Capacidad para generar datos aleatorios (por ejemplo, siguiendo distribuciones Gaussianas o normales) y visualizarlos en histogramas o gráficos de dispersión con Matplotlib.
  • Regresiones: Aplicación de modelos clásicos de aprendizaje supervisado, logrando regresiones lineales con SciPy, polinómicas con NumPy, o regresiones múltiples robustas usando Pandas y Scikit-learn.
